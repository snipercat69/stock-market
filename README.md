# stock-market

Discord bot that delivers daily stock market updates using Yahoo Finance API. Tracks 100+ assets including indices, crypto, commodities, tech stocks, finance, healthcare, defense, retail, and more. Posts to Discord at 4 PM ET on weekdays.

## Topics
python, discord, bot, stocks, finance, yahoo-finance, market-data

## Setup

```bash
# Clone the repo
git clone https://github.com/snipercat69/stock-market.git
cd stock-market

# Install dependencies
pip install -r requirements.txt
```

## Usage

```bash
python3 stock-market.py
```

## About

Built with [OpenClaw](https://github.com/openclaw/openclaw) — an AI agent framework. This app runs as a Discord bot / scheduled automation powered by OpenClaw's cron and channel integration system.
